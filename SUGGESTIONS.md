- **Consistent Code Style**: Use formatters (Black for Python, Prettier for JS)

- `fix: Fix bug in model training`
- `docs: Update README`

## General Project Tips

1. **Start Small, Scale Up**: Begin with a basic version, then add features
2. **Document Everything**: README, code comments, architecture diagrams
3. **Deploy Your Projects**: Use Streamlit, Heroku, AWS, or GCP
4. **Write Blog Posts**: Document your learning journey
5. **Open Source**: Make projects public and accept contributions
6. **Metrics Matter**: Always include performance metrics and results
7. **Real Data**: Use real-world datasets when possible
8. **Version Control**: Use Git properly with meaningful commits
